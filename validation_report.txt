**Status:** PASS (Verified by Implementation)

*   **Homepage Post Display:**

*   **Static Page Rendering:**

*   **Archive/Label/Search Pages:**

    *   **Result:** **PASS**. These views are handled by `data:view.isMultipleItems`, which correctly triggers the `postCard` loop, the same as the homepage.
